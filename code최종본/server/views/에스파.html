  • 소속사 : SM 엔터테인먼트
  • 데뷔 : 2020-11-17
  • 팬덤 : MY(마이)
"Be MY æ! 안녕하세요, aespa입니다!"
aespa는 2020년 11월 17일에 데뷔한 SM엔터테인먼트 소속 4인조 다국적 걸그룹이다.
2020년에 데뷔한 그룹 중 굉장히 독보적인 성적을 내고 있는 4세대 K-POP 차세대 대표주자 중 하나인 걸그룹이다. 그룹명 'aespa (에스파)'는 ‘아바타 X 익스피리언스 (Avatar X Experience)’를 표현한
‘æ[읽는법]’와 양면이라는 뜻의 영단어 ‘aspect’를 결합해 만든 이름이다.
그룹명을 æspa로 표기할 때도 있다.
