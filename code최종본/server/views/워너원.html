Wanna One은 2017년 8월 7일에 데뷔한 대한민국의 11인조 보이그룹이다. 국내 가수 최초로 고척 스카이돔에서 데뷔하였으며 2019년 1월 27일까지 프로젝트 그룹으로서 활동했다.
