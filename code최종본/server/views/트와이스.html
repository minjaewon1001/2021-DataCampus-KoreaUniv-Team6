“ONE IN A MILLION! 안녕하세요, TWICE입니다!!”
TWICE는 2015년 10월 20일에 데뷔한 JYP엔터테인먼트 소속
한국인 5명, 일본인 3명, 대만인 1명으로 이루어진 대한민국의 9인조 다국적 그룹이다.
3세대 K-POP 걸그룹의 세대교체를 이끈 주역 중 하나로 평가를 받는 그룹이기도 하며,
데뷔 초부터 지금에 이르기까지 폭발적인 성적을 바탕으로 3세대 K-POP을 대표하는 정상급 그룹 중 하나로 꼽히고 있다.
시그니처 사운드는 멤버들이 다 함께 "TWICE!"라고 외치는 사운드다.
